

##  Постановка задачі

У цьому домашньому завданні розглядається задача оптимізації потоку від загального джерела через термінали, склади та магазини до спільного приймача.

У такій постановці задачі таблицю з результатами потоків **між терміналами та магазинами**, як це зазначено у конспекті, заповнити неможливо, оскільки потоки перемішуються на складах.  

Можна було б окремо шукати максимальний потік між кожним терміналом та кожним магазином, але така постановка суперечить ідеї аналізу транспортної мережі.
